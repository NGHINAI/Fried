# Fried — TikTok Go-To-Market Playbook

Research-backed (2026). How Cal AI, Umax, Rizz, Locket, Saturn, Gas actually blew up — applied to Fried. Full sourced research: see chat / the research agent output.

## The unfair advantage
Fried's customer is **already on TikTok feeling the exact pain we solve** (brain rot). And the **"aura points" / "brain rot quiz"** trend is a built-in distribution language ("ways to lose aura" hit 26M views). Our **Fried Score + Brain Age + Archetype card** is our "Umax rating screen" — the single most viral object we can build. **The card does the marketing; users make the content.**

## The model (do these in ORDER, never simultaneously)
1. **Organic volume first** — find a winning video *template* with ~$0.
2. **Concentrate** — once one format over-indexes, run it into the ground.
3. **Paid second** — Spark-boost ONLY proven winners (40–60% better CPA; TikTok CPI is $0.50–$1.80, the cheapest install source).

> Cal AI: "social proof is manufactured through repetition, not a single hero video." 1,000+ videos, 12+ accounts. Locket: 52 videos/day across 26 accounts → 298M views. Rizz: $50 to two creators → 200K downloads overnight.

## What videos to make (top 3 formats for Fried)
1. **Green-screen the result (the reveal)** — react to the Fried Score + archetype card. The absurd number is the payoff. *#1 format.*
2. **Screen-record the app** — 90-sec test sped up → score reveal. Silent, native, lofi. "Demonstrable in 3 seconds."
3. **Rating strangers' brains / street interview** — walk up, have them take it, react to scores. High novelty → high shares + duets.

Supporting: POV/relatable ("me pretending I don't have brain rot" → cut to app), confessional ("I tested how fried my brain is"), aura-points crossover, duet/stitch bait, before/after.

## The 2026 algorithm (engineer toward these)
- **Completion ≥70%** + **rewatch 20–30%** + **shares > comments > saves > likes**. Followers DON'T matter (every video judged on its own).
- Length sweet spots: **5–7s** (test→score) and **27–35s** (funny/feature).
- Export CLEAN (no watermark), native/raw beats polished, **vary every clip** across accounts.
- Captions/hashtags are a SEARCH surface now: repeat the hook in caption + 2–3 niche tags (#brainrot #aurapoints #screentime #fyp).

## Conversion funnel (the App Store link constraint!)
- **Personal accounts CANNOT link the App Store.** Fix: **TikTok Business account** for the hero handle (clickable store link) + a **landing page that smart-redirects iOS→App Store**.
- **"Comment FRIED → auto-DM the link"** via **ManyChat × TikTok** — turn it ON *before* posting (only fires on new comments; also juices comment count = ranking signal).
- **Pinned comment** with "getfried.app" as copy-paste fallback. Say the name in the video (TikTok search is a real install channel — make "Fried" findable).
- **ASO:** first 3 screenshots = the score/brain-age/archetype card (continuity from video). Apple now indexes screenshot caption text → put "brain rot test, brain age, aura, fried score" in captions. Keep 4.0★+ (review prompt at the score-reveal delight peak — already built).

## Distribution channels, by ROI
1. **Founder/brand account, high volume** — free, the engine.
2. **Micro-creators** (<50K followers who already hit 1–2 viral videos): **$50–$100/video** base + **$1 CPM** bonus. (Rizz's $50 arbitrage.)
3. **Pay-per-view clipping bounties** — **Whop Content Rewards / Vyro**: fund a cap, clippers post your swipe-pack, pay **~$1–$3 per 1,000 verified views**. Performance-only top-of-funnel.
4. **Spark Ads** on proven organic winners only — start $30–50/day, optimize for "start trial"/first event (cleaner iOS signal).
5. *(Aggressive, gray-area, ToS risk: multi-account "farms" with anti-detect tooling. Mentioned for completeness — lead with 1–4.)*

## 30-day plan
- **Pre-flight:** Business hero account + 5–8 faceless niche handles (@howfriedru, @brainrotchecks, @auraloss…). Landing page (smart redirect + pixel). ManyChat keyword DM. Submagic + CapCut. Swipe file of 30–50 viral aura/brain-rot/face-rating videos. One-tap "share my Fried Score" (score baked into the card).
- **Week 1 — seed:** warm accounts 24–48h (no posting), then 2–3 native posts/account/day, run all formats × hooks (exploration). ~50–80 videos. DM 50–100 micro-creators/day.
- **Week 2 — read + concentrate:** find the 3–5 videos with completion >55% / rewatch >20% / shares spiking. Kill the rest. Shift 70% of volume to the 2 winners. Launch a small Whop bounty.
- **Week 3 — hero push + paid:** street-interview + #friedchallenge duet-bait. Turn on Spark Ads on proven winners ($30–50/day). Scale cluster to 15–25 accounts.
- **Week 4 — compound:** referral/promo codes for creators (attribution + step-change). Push the in-product share loop. Ride the App Store ranking flywheel. Cut anything >$5 CPI.
